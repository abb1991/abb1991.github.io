      • L Train Bingo is a Rails/React.js web app that allows users to create real-world bingo cards and play with their friends. All work on this app was a solo effort.

    • Shortstack

    • Message In A Bottle

      • Message In A Bottle is a journaling web app that allows users to send and receive anonymous journal entries. The app is centered around creating opportunities for users to create meaningful and raw content without worrying about how they're being perceived by the world at large, while simultaneously writing for an audience. This project was completed over the a week by three developers.

      • Battleboat is a three-day team of five project to create a clone of Battleship. It uses a ruby on rails framework and javascript to create real-time turn based play between users across a network.

      • This solo project creates a playable game of standard Hearts. Using a Sinatra framework and an ERB frontend.
